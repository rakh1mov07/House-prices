# House Prices — Предсказание стоимости жилья

Проект по машинному обучению: исследование датасета с характеристиками домов и построение регрессионных моделей для предсказания цены продажи.

---

## Описание задачи

- **Тип задачи:** регрессия  
- **Целевая переменная:** `SalePrice` — цена продажи дома (USD)  
- **Источник данных:** датасет, вдохновлённый *Kaggle House Prices*

---

## Описание датасета

| Признак           | Описание                                      |
|------------------|-----------------------------------------------|
| OverallQual      | Качество материалов и отделки (1–10)          |
| GrLivArea        | Жилая площадь над землёй (кв. футы)           |
| YearBuilt        | Год постройки                                |
| TotalBsmtSF      | Общая площадь подвала (кв. футы)              |
| GarageCars       | Вместимость гаража (количество машин)         |
| FullBath         | Количество полных ванных комнат               |
| Fireplaces       | Количество каминов                            |
| NeighborhoodScore| Рейтинг района (1–7)                          |
| LotArea          | Площадь участка (кв. футы)                    |
| MasVnrArea       | Площадь облицовки (кв. футы)                  |
| SalePrice        | Цена продажи (целевая переменная)             |

**Пропущенные значения:**
- `TotalBsmtSF` — 37  
- `GarageCars` — 81  
- `MasVnrArea` — 8  

---

## Методология

### 1. Разведочный анализ (EDA)

- Анализ распределения целевой переменной — выявлена правая асимметрия  
- Корреляционный анализ признаков  
- Визуализация зависимостей (scatter plot, boxplot)  

### 2. Предобработка данных

- Лог-трансформация `SalePrice` для нормализации распределения  
- Заполнение пропусков медианными значениями  
- Масштабирование признаков (`StandardScaler`)  
- Разделение данных: `Train/Test = 80/20` (`random_state=42`)  

### 3. Модели и подбор гиперпараметров

Использован `GridSearchCV` с 5-fold кросс-валидацией.

| Модель            | Параметры                                   | Лучшие значения        |
|------------------|--------------------------------------------|------------------------|
| Ridge            | alpha                                      | 1                      |
| Lasso            | alpha                                      | 0.001                  |
| Random Forest    | n_estimators, max_depth, min_samples_split | 200, None, 5           |
| Gradient Boosting| n_estimators, learning_rate, max_depth     | 200, 0.05, 3           |

---

## Результаты

| Модель            | RMSE (log) | MAE (log) | R²    |
|------------------|-----------|----------|-------|
| Ridge            | 0.0923    | 0.0716   | 0.806 |
| Lasso            | 0.0926    | 0.0717   | 0.805 |
| Gradient Boosting| 0.0965    | 0.0743   | 0.788 |
| Random Forest    | 0.1100    | 0.0854   | 0.725 |

**Лучшая модель:** Ridge Regression  
- CV RMSE (5-fold): `0.0884 ± 0.0025`

---

## Ключевые выводы

- Наиболее значимые признаки:
  - `GrLivArea` (корреляция: 0.59)  
  - `OverallQual` (корреляция: 0.49)  

- Лог-трансформация целевой переменной:
  - снизила асимметрию с `0.12` до ~`0.01`
  - улучшила качество линейных моделей  

- Ridge Regression показала наилучший результат (`R² = 0.806`), что указывает на преимущественно линейные зависимости  

- Gradient Boosting занял второе место и может быть улучшен за счёт feature engineering  

- Остатки модели распределены нормально вокруг нуля, что указывает на корректную подгонку
